Screen Title: SCREWING UNSCREWING STATION

Screen Purpose: 
This screen allows operators to monitor and control the screwing/unscrewing process for multiple screw blowers in real time.

Layout:
- Type: grid
- Columns: 4
- Rows: 8
- Orientation: landscape

Color Scheme:
- Primary: #FFD600 (yellow)
- Secondary: #424242 (dark gray)
- Background: #23272A (very dark gray)
- Text: #FFFFFF (white)
- Accent: #FF0000 (red), #FFC300 (yellow), #BDBDBD (light gray), #000000 (black)

Header:
- Title: "SCREWING UNSCREWING STATION" (centered, large, white)
- Subtitle: "CM CONTROL" (top right, yellow)

Table Columns:
1. CM NAME (yellow, left-aligned)
2. STATUS (yellow, with colored squares: red/yellow)
3. COMMAND (yellow, with two buttons: START, RESET)
4. STEP TIME (yellow, with empty boxes)

Rows (repeat for 8 screw blowers):

For each row:
- Label: "SCREW BLOWER"
- Status Indicator: colored square (red or yellow)
- START Button: gray background, black text
- RESET Button: gray background, black text
- Step Time Box: empty, outlined in yellow

Required Elements:
- 8x Label: "SCREW BLOWER"
- 8x Status Indicator: (6 red, 1 yellow, 1 red) [as per image]
- 8x START Button
- 8x RESET Button
- 8x Step Time Box

Element Details:
- Status Indicator: square, 32x32px, color: #FF0000 (red) or #FFC300 (yellow)
- START Button: rectangle, 100x40px, color: #BDBDBD (light gray), text: "START", text color: #000000
- RESET Button: rectangle, 100x40px, color: #BDBDBD (light gray), text: "RESET", text color: #000000
- Step Time Box: rectangle, 100x40px, border: 2px solid #FFD600 (yellow), background: transparent

Interactions:
- Clicking START or RESET sends a command to the corresponding screw blower.

Data Fields:
- CM Name (string)
- Status (enum: red, yellow)
- Step Time (number, seconds)

Accessibility:
- High contrast for all text and indicators.
